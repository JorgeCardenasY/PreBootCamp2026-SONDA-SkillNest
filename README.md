# Pre-Bootcamp Data Science - SkillNest

## Descripción

Este repositorio contiene ejercicios y prácticas realizados en el contexto del **Pre-Bootcamp de Data Science** organizado por **SkillNest**, auspiciado por **SONDA**. El objetivo es preparar a los participantes con conocimientos fundamentales en programación y análisis de datos antes del inicio del bootcamp principal.

Los ejercicios incluyen prácticas en Python, manipulación de estructuras de datos como listas y diccionarios, y la creación de funciones básicas e intermedias.

## Contenido del Repositorio

- `funciones_basicas_1.py`: Ejercicios básicos de funciones en Python.
- `funciones_basicas_2.py`: Continuación de ejercicios básicos.
- `funciones_intermedias_1.py`: Ejercicios intermedios con iteración de diccionarios y listas.
- `Hola_Mundo/`: Carpeta con ejercicios iniciales, incluyendo bucles y estructuras básicas.
- `Reconocer_Python_(Practica)/`: Prácticas para reconocer y familiarizarse con Python.
- `Tarea_1/`: Primera tarea del pre-bootcamp.
- `Tarea_2/`: Segunda tarea del pre-bootcamp.

## Requisitos

- Python 3.x instalado en el sistema.
- Recomendado: Entorno virtual (venv) para ejecutar los scripts.

## Cómo Ejecutar

1. Clona o descarga este repositorio.
2. Navega a la carpeta del proyecto.
3. Ejecuta los archivos Python con el comando:
   ```
   python nombre_del_archivo.py
   ```

Por ejemplo:
```
python funciones_intermedias_1.py
```
